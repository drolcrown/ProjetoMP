Trabalho 2

Usando programação voltada a testes.

Antes de começar criar o código são pensados os testes para que o código funcione corretamente
ao ser usado por um usuário. Os testes são criados primeiro que o programa principal.

Os testes são testados com o GoogleTest.
   	Para se executar os testes corretamente deve-se usar a palavra make para executar o Makefile criado. Depois disso
   será criado os objects, na pasta OBJ, e um executável na pasta SRC. É necessário que a execução o executável criado e os 
   testes serão rodados automáticamente.
